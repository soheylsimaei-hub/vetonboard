# Vet On Board

The world, seen through the eyes of a veterinarian.

Cinematic editorial media platform created by Dr. Soheyl Simaei.

## Status

Pre-launch — coming-soon mode. Vol. 01, Lisbon, 2026. Public launch targeted for Q2 2026.

## Stack

Static HTML/CSS/JS. No build step. Deployed via Vercel.

## Local preview

Open `index.html` directly in a browser, or serve the folder:

```
python3 -m http.server 3000
```

Then visit http://localhost:3000.

## Structure

- `index.html` — the homepage (single page, all sections inline)
- `assets/` — logo and other static assets

## Deploy

Pushes to `main` auto-deploy to Vercel.
